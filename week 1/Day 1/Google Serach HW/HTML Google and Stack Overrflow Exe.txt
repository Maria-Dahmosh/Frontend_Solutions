<!DOCTYPE html>: 
An instruction to the browser regarding the version of HTML the page is written in.
a decleration thet HTML 5 is used.

<html>:
Used to specify the roots of HTML page. 
The <html> tag informs the browser that this is an HTML document. It is the second outer container for everything in an HTML document, and requires a beginning and ending tag.

<head>:
The <head> tag in HTML is used to contain metadata (data about data). It is used between the<html> and <body> tags.
The content of the head is not displayed in the browser when the page loads. It only contains HTML document metadata, which specifies information about the HTML document, requires a beginning and ending tag.

<body>
Specifies the main content of an HTML document that appears on the browser. It can contain headings, text, paragraphs, photos, tables, links, videos, etc.
The <body> tag must come after the <head> tag. This tag is essential for all HTML documents and should only be used once throughout the document, requires a beginning and ending tag.
